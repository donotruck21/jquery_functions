jQuery Practice

.hide/.show

Click the ".hide" button to hide this paragraph! Click the ".show" button to make it reappear!

.toggle

Click the ".toggle" button to toggle between the .hide and .show functions! Click once to make the text disappear, and click again to make it reappear!

.slideDown/.slideUp

The .slideUp and .slideDown functions work exactly like the .hide and .show functions, except with a cool sliding feature! Check it out!

.slideToggle

The .slideToggle function works exactly like the .toggle function, except with a cool sliding feature! Check it out!

.fadeOut/.fadeIn

The .fadeOut and .fadeIn functions work exactly like the .hide and .show functions, except with a cool fading feature! Check it out!

.addClass

The .addClass jQuery funtion will add specified class(es) to each matched element. You can add multiple classes at a time. Click on the ".addClass" button to add class "red" and class "bold" to the text! Click on the ".removeClass" button to return text to default color black.

.css

The .css jQuery function will set the one or more properties for any matched elements. Click on the ".css" button to set all of the h2 headings on this page to blue!

.after

The .after jQuery function is a manipulation that inserts content after each element in the set of matched elements. Click on the ".after" button to insert another another paragraph element after this paragraph!

.before

The .before jQuery function is a manipulation that inserts content before each element in the set of matched elements. Click on the ".before" button to insert another another paragraph element before this paragraph!

.append

The .append jQuery function is a manipulation that inserts content to the end of each element in the set of matched elements. Click on the ".append" button to insert another another paragraph element at the end of this paragraph!

.attr

The .attr jQuery function gets the value of an attribute for a matched element. Hover over the image to see the "alt" attribute.

.html

The .html jQuery function is a manipulation that sets the HTML contents of each element in the set of matched elements

.text

The .text jQuery function can work as a getter or as a setter. In most cases, you will be using the .text function as a setter. Click on the ".text" button to set this text to a specified text!

.val

The .val jQuery function also can work as a getter or a setter. In this case, we are going to use it as a setter. The .val function will get the current value of the first element in the set of matched elements. Type something into the input box and click ".val" to return the same value.

*Notice the text that appears when you click into the text box! The .focusin/.focusout event handlers allow text to appear and disappear as you enter and leave the text box!
